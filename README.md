# 🚚 Smart Supply Chain AI

## 📌 Problem

Modern supply chains handle thousands of shipments daily, but disruptions like traffic, weather, and delays are often detected too late.
This leads to delivery delays, increased costs, and inefficiencies.

---

## 💡 Solution

Smart Supply Chain AI is a real-time monitoring and prediction system that:

* Tracks shipments live on a map
* Detects delays and risks early
* Suggests rerouting and optimization
* Helps logistics teams make faster decisions

---

## 🚀 Features

### 📍 Live Tracking

* Real-time truck movement using map visualization

### ⚠️ AI Alert System

* Detects:

  * Delay risks
  * Weather disruptions
* Generates alerts automatically

### 🔄 Smart Actions

* **Reroute** for delays
* **Optimize** for risk conditions

### 📊 Dashboard

* Total shipments
* Delayed shipments
* At-risk shipments

### ✅ Resolved Alerts

* View resolved issues
* Clear history option

### 🔐 Login System

* Simple authentication (demo-based)

---

## 🛠️ Tech Stack

**Frontend**

* HTML
* CSS
* JavaScript
* Leaflet.js (map visualization)

**Backend**

* Python
* Flask

**Database**

* SQLite

---

## 🧠 How It Works

1. Trucks move on predefined routes (simulation)
2. System periodically generates AI-based alerts
3. Alerts appear in dashboard
4. User can:

   * Reroute (for delays)
   * Optimize (for risks)
5. Resolved alerts are stored for tracking

## ▶️ How to Run Locally

### 1. Clone Repository

```
git clone https://github.com/yourusername/smart-supply-chain-ai
cd smart-supply-chain-ai
```

### 2. Install Dependencies

```
pip install flask flask-cors
```

### 3. Run Backend

```
python app.py
```

### 4. Open Frontend

* Open `index.html` in browser
  *(or use Live Server for best experience)*
